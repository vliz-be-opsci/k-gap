# Components

K-GAP is composed of specialized services. Use this chapter to navigate component-level reference documentation:

- [GraphDB Component](./components/graphdb.md)
- [Jupyter Component](./components/jupyter.md)
- [Sembench Component](./components/sembench.md)
- [LDES Consumer Component](./components/ldes-consumer.md)

## Alternative Triple Stores

K-GAP also supports the following triple stores as drop-in alternatives to GraphDB:

- [Oxigraph Component](./components/oxigraph.md) — lightweight Rust-based RDF store
- [Apache Jena Fuseki Component](./components/fuseki.md) — classic Java SPARQL server with named datasets
- [Virtuoso Component](./components/virtuoso.md) — high-performance multi-model database with named-graph isolation
